# Data Description for Retal_Sales_Analysis

## Dataset Source
The dataset used in this project is publicly available.

Dataset: Superstore Sales Dataset
Source: Kaggle

If applicable:

The data is publicly available at: https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting/data
A sample copy is included in /data/train.csv for reproducibility.

## Dataset Description
Covering a four-year period from 2015 to 2018, the dataset comprises approximately 9,800 sales transactions.

Key columns:
- Order ID
- Order Date
- Ship Date
- Ship Mode
- Customer ID
- Customer Name
- Segment
- Region
- Country
- City
- State
- Postal Code
- Region
- Product ID
- Category
- Sub-Category
- Product Name
- Sales
